# Buyer Portal

A full-stack real estate buyer portal where users can register, login, upload properties, manage favourites, and customize their profile.

## Live Demo

- **Frontend:** https://buyer-portal-alpha.vercel.app
- **Backend Health Check:** https://buyer-portal-backend-6uhc.onrender.com/api/health

> **Note:** The backend is hosted on Render's free tier and may take 30-60 seconds to wake up on the first request if it has been inactive. Please wait a moment if the initial load is slow.

## Tech Stack

- **Frontend:** React.js, Tailwind CSS, Vite
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL (Supabase)
- **Image Storage:** Cloudinary
- **Auth:** JWT (JSON Web Tokens)
